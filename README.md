# Hi, I'm Ayush Gangwar

Software Engineer building backend systems, AI infrastructure, and developer tooling.

Most of my work involves building production systems, but I'm broadly interested in how software behaves under real-world operational constraints.

The questions that keep me interested are simple:

- What changes under concurrency?
- Where does state live throughout execution?
- How do systems recover from failures?
- Where do bottlenecks emerge?
- How does architecture evolve as complexity grows?

I prefer systems that remain reliable, observable, maintainable, and easy to reason about.
